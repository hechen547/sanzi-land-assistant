# 三资图斑辅助工具使用说明

## 启动

双击项目目录中的 `启动软件.bat`。

也可以在 PowerShell 中运行：

```powershell
uv run sanzi-photo-tool
```

## 推荐操作顺序

软件包含六个主工作区，顺序为：

1. 图斑下载
2. 航线规划
3. 照片处理
4. 图斑整理
5. 三资上传
6. 地图软件

### 图斑下载

1. 登录三资平台并进入数据采集地图。
2. 选择需要的村庄。
3. 在平台左侧勾选需要的“工作进度”，等待图斑显示在地图上。
4. 点击右上角“下载当前已显示图斑 KML”。
5. 选择保存位置。

该功能只下载地图上当前已经显示出来的工作进度图斑。

### 照片处理

1. 左侧选择照片目录并扫描，勾选需要处理的照片。
2. 中间切换“原图／处理后”，实时检查水印效果。
3. 右侧设置水印内容、字号、颜色和输出目录；批量命名固定从 `A1` 开始。
4. 点击底部“开始处理照片”。

### 图斑整理

1. 选择一个或多个 KML 文件。
2. 软件自动使用“照片处理”页面中已勾选的照片。
3. 设置最近匹配距离、复制／移动模式和输出目录。
4. 先点击“预分析匹配结果”，确认统计和匹配明细。
5. 点击“确认并开始整理”。

### 航线规划

- 在软件内打开项目根目录的 `index.html`。
- 可规划航点、导入 KML/KMZ，并导出航线文件。
- 地图与外部脚本需要联网加载。

### 三资上传

1. 点击“登录平台并自动获取”，在内置页面完成三资平台登录。
2. 登录后点击“登录完成，自动获取”，软件自动读取 Token 和行政区信息。
3. 选择已经按图斑编号整理好的照片根目录。
4. 先执行“上传前检查”，确认可上传、跳过和失败明细。
5. 确认无误后点击“确认并开始上传”。
6. 上传结束后可导出 CSV 日志。

账号和密码仅在平台登录页面输入，软件不会保存。

### 地图软件

- 在软件内打开项目根目录的 `gps_map.html`。
- 支持加载 KML 图层、搜索和查看照片点位。

## 输出原则

- 水印和批量命名默认复制到新目录，不覆盖原图。
- 图斑整理默认复制照片；移动模式会在执行前再次确认。
- 文件重名时自动增加 `_1`、`_2` 等后缀。
- 图斑整理会生成所有图斑目录，以及：
  - `未匹配图斑`
  - `无照片图斑.kml`
  - `未匹配照片.kml`
  - `整理结果.csv`

## 图斑匹配

- `最近匹配距离` 为 0 时，只匹配图斑内部和边界上的照片。
- 设置大于 0 的距离后，未直接命中的照片可以匹配到阈值内最近的图斑。
- 没有 GPS 的照片不会参与图斑整理。
