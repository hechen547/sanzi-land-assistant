# 三资辅助软件 1.2 使用说明

## 启动

双击项目目录中的 `启动软件.bat`。

也可以在 PowerShell 中运行：

```powershell
uv run sanzi-photo-tool
```

## 推荐操作顺序

软件包含七个主工作区，顺序为：

1. 下载平台图斑
2. 制作无人机航线
3. 给照片加水印
4. 按图斑整理照片
5. 上传照片到平台
6. 查看照片地图
7. 使用说明

### 下载平台图斑

1. 登录三资平台并进入数据采集地图。
2. 选择需要的村庄。
3. 在平台左侧勾选需要的“工作进度”，等待图斑显示在地图上。
4. 点击右上角“保存当前看到的图斑”。
5. 选择保存位置。

该功能只下载地图上当前已经显示出来的工作进度图斑。

### 给照片加水印

1. 左侧选择照片文件夹，点击“读取照片”，再勾选需要处理的照片。
   - 照片较多时左侧会分页显示，每页最多 500 张；全部照片仍默认参与处理。
   - 软件会自动缓存已经读取的拍摄时间、GPS和尺寸，再次读取相同照片会更快。
   - EXIF中没有GPS时，软件会继续尝试读取无人机照片中的XMP定位信息。
2. 中间切换“原图／水印效果”，检查生成后的样子。
3. 右侧设置水印内容、字号、颜色和保存位置；照片编号固定从 `A1` 开始。
4. 选择处理速度：SSD 使用“快速处理”，老旧电脑、机械硬盘或 U 盘使用“兼容处理”。
5. 点击底部“生成新照片”。

### 按图斑整理照片

1. 选择一个或多个图斑文件。
2. 在本页选择需要整理的照片文件夹，点击“读取照片”。
3. 软件会统计有定位和没有定位的照片；只有带定位的照片可以和图斑进行判断。
4. 选择原照片是否保留。
5. 如需给第一轮没有照片的图斑补充附近照片，保留“为空图斑补充附近照片”并设置补充距离。该选项只在“保留原照片”模式下可用。
6. 选择整理速度：SSD 和性能较好的电脑使用“快速整理”；老旧电脑、机械硬盘或 U 盘使用“兼容整理”。两种模式都保留照片内容、EXIF 拍摄信息和 Windows 文件时间。
7. 选择整理结果的保存位置。
8. 先点击“先看看整理结果”，确认照片会被分到哪里。
9. 点击“确认后开始整理”。

整理时会按照 KML 中的全部有效地块编码建立文件夹。没有照片的图斑也会保留空文件夹，方便上传前检查是否漏拍；重复编码会合并为同一个文件夹。

### 制作无人机航线

- 在软件内打开项目根目录的 `index.html`。
- 可规划航点、导入 KML/KMZ，并导出航线文件。
- 地图与外部脚本需要联网加载。

### 上传照片到平台

1. 点击“打开登录页面”，在软件内完成三资平台登录。
2. 登录后点击“我已登录，继续”，软件会自动确认登录状态。
3. 需要更换账号时点击“切换账号”；只想清除登录状态时点击“退出登录”。
4. 选择本次上传对应的 KML 图斑文件。
5. 选择已经按图斑编号整理好的照片根目录。
6. 软件会核对当前登录地区、KML 图斑编号和照片文件夹编号；任一不一致都会阻止整批上传。
7. 点击“先检查哪些照片能上传”，查看准备上传、暂不上传和错误明细。
8. 确认无误后点击“确认上传照片”。
9. 上传结束后可保存结果表格。

上传检查以 KML 全部图斑为标准。文件夹名称必须完全等于完整地块编码；没有照片、缺少文件夹、编码异常、平台查不到、已经上传和地区不一致都会在右侧分别显示。

每次检查或上传前，软件都会从平台网页重新读取最新 Token。若平台返回401，软件会立即停止并要求重新登录，不会继续检查剩余图斑。

账号和密码仅在平台登录页面输入，软件不会保存。

### 查看照片地图

- 在软件内打开项目根目录的 `gps_map.html`。
- 支持加载 KML 图层、搜索和查看照片点位。

## 输出原则

- 水印和批量命名默认复制到新目录，不覆盖原图。
- 图斑整理默认复制照片；移动模式会在执行前再次确认。
- 文件重名时自动增加 `_1`、`_2` 等后缀。
- 图斑整理会为 KML 中全部有效地块编码建立文件夹。未匹配照片不会复制，没有照片的图斑保留为空文件夹。
- 整理结果还会生成：
  - `无照片图斑.kml`
  - `未匹配照片.kml`
  - `图斑外距离匹配照片清单.txt`
  - `图斑照片分类工作日志.txt`
  - `整理结果.csv`

## 照片如何归入图斑

- “图斑外允许距离”为 0 米时，只整理确实位于图斑内部或边界上的照片，最稳妥。
- 设置大于 0 米后，图斑附近的照片也可能归入距离最近的图斑。
- 没有定位信息的照片不会参与整理，并会单独统计。
