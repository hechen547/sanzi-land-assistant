# 三资图斑辅助工具

Windows 桌面照片处理工具，提供：

- 扫描照片并读取拍摄时间、GPS 和 EXIF 信息
- 自定义左下角水印内容、字号和颜色并批量输出
- 按拍摄时间或文件名批量编号
- 手动解析一个或多个 KML，按图斑归属整理照片
- 导出无照片图斑、未匹配照片和 CSV 处理报告

界面采用六个主工作区：

- **图斑下载**：登录三资平台后，下载地图上当前已显示的工作进度图斑 KML。
- **航线规划**：内嵌 `index.html` 航线规划器。
- **照片处理**：照片选择、水印预览、批量命名和输出集中在同一页面。
- **图斑整理**：独立完成 KML 导入、预分析和按图斑整理。
- **三资上传**：内置平台登录、上传前检查、批量上传和 CSV 日志。
- **地图软件**：内嵌项目根目录中的 `gps_map.html`。

## 开发运行

项目使用 `uv` 管理独立环境，不使用全局 Python 包。

```powershell
uv sync --extra dev
uv run sanzi-photo-tool
```

依赖安装需由项目所有者明确确认后执行。
