# 三资辅助软件 1.2 发布说明

发布日期：2026年6月20日

## 主要更新

- 修复上传页登录与 Token 获取逻辑，不再要求用户手动进入下载页初始化。
- 上传前检查和正式上传增加真实进度、当前任务提示和停止按钮。
- 检查完成后自动保存检查结果表，并确保右侧正常显示结果。
- 完善图斑整理的 UTM 距离计算、异常图斑修复和空图斑附近照片补充。
- 水印、图斑分析和照片整理任务增加进度与取消支持。
- 优化普通地区和山区的距离设置说明，降低照片误分风险。
- 新增完整“使用说明”页面，包含操作步骤、参数建议和常见错误处理。
- 改进上传状态说明、登录切换和大量数据处理体验。

## 使用提醒

- 普通地区建议把照片允许偏离距离设为 10～20 米。
- 山区可先尝试 20～30 米，超过 50 米容易把照片归入相邻图斑。
- 上传前先查看检查结果，只有“可上传”状态的照片才会正式上传。
- 平台出现 `timed out` 或 HTTP 502 通常是网络或平台繁忙，可稍后重试。
- 广泛对外分发前建议为安装程序购买 Windows 代码签名证书。
